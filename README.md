# OOP-Concepts

A reference guide to core Object-Oriented Programming concepts, organized
into three parts:

- [1 Core Building Blocks](OOP%20Concepts/1%20Core%20Building%20Blocks/README.md) — classes, objects, attributes, methods, constructors
- [2 The 4 Pillars That Make OOP Powerful](OOP%20Concepts/2%20The%204%20Pillars%20That%20Make%20OOP%20Powerful/README.md) — encapsulation, abstraction, inheritance, polymorphism
- [3 Object Relationships: How Classes Work Together](OOP%20Concepts/3%20Object%20Relationships%20How%20Classes%20Work%20Together/README.md) — association, aggregation, composition, dependency
